Bing Image Picker adds a 🖼 button to the note editor: search Bing Images for the note's word and pick from a 3×3 grid. The chosen image is saved to your collection and written to a field.

On first use per note type, a dialog asks which field to search from, which to put the image in, and append vs. overwrite. The mapping is editable later via Tools → Bing Image Picker. Safe-Search is off. Source: github.com/steindekker/bing-image-picker
